Python para Engenheiros

Curso Prático

Download .zip Download .tar.gz View on GitHub

Material do mini-curso criado depto. de desenvolvimento de software da ESSS para engenheiros e demais profissionais/estudantes de áreas exatas. O material do curso apresentado aqui é bastante direto, e provavelmente não é a melhor introdução ao Python por si só, mas pode ser utilizado junto com outros tutoriais para encurtar o tempo necessário que o estudante estará apto a criar programas com foco em desenvolvimento científico e engenharia.

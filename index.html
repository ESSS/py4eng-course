Python for Engineers

Crash Course

View project onGitHub

This was a crash course of Python taught by ESSS software dev. department to the company CFD/FEA engineers, since they were writing a lot of code (using different languages and environments) to aid their daily activities. The course material presented here is very straightforward, and probably it's not the best Python introduction per se, but could go along with other tutorials to shorten the necessary time a student need to be able to create usefull engineering/scientific tools using Python.

Chapter 1 - Python Basics

Chapter 2 - The Scipy Stack

Chapter 3 - Optimization Problem using ANSYS Mechanical

Chapter 3a - Regular Expressions

Chapter 4 - Building Graphical Applications - the fast and easy way

It's expected this material to be in constant improvement.
